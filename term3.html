The CS141 Perpetual Haskelling Initiative is an optional, collaborative Haskell programming project which will run in Week 4 of Term 3 (exact dates to be confirmed). By participating, we will:

  • Collaboratively develop a realistic software application in Haskell (and other languages, as required) and gain experience with Haskell programming, libraries, etc.
  • Gain applied software engineering experience through the use of version control, code reviews, continuous integration, etc. in the same way as you would when working in the software industry.

If you would like to participate in the discussion over what the exact project we will work on should be, feel free to join in. Your thoughts or ideas would be most appreciated!
